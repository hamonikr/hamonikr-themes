## hamonikr-themes

 * 하모니카 OS 테마
 * plank theme, McMojave gtk3(https://www.gnome-look.org/p/1275087/) 등을 적용한 테마

## License

 * GPL-3+

## Install

### HamoniKR (>=6.0)
```
sudo apt install hamonikr-themes
```

### Ubuntu (>=20.04)
```
wget -qO- https://update.hamonikr.org/add-update-repo.apt | sudo -E bash -

sudo apt install hamonikr-themes
```
