## HamoniKR OS Theme

 * Support gtk4, gtk3, cinnamon, mate, gnome
 * Include cursor, plank
 * Light and Dark
 * Include HamoniKR icon

## License

 * GPL-3+

## Install

### HamoniKR (>=6.0)
```
sudo apt install hamonikr-themes
```

### Ubuntu (>=20.04)
```
wget -qO- https://update.hamonikr.org/add-update-repo.apt | sudo -E bash -

sudo apt install hamonikr-themes
```
