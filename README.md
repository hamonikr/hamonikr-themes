## HamoniKR OS Theme

 * Support gtk4, gtk3, cinnamon, mate, gnome
 * Include cursor, plank
 * Light and Dark
 * Include HamoniKR icon

## License

 * GPL-3+

## Install

### Ubuntu, Debian, LinuxMint...
```
wget -qO- https://pkg.hamonikr.org/add-hamonikr.apt | sudo -E bash -

sudo apt install hamonikr-themes
```
